# MCP服务器配置项目

这是我的MCP (Model Context Protocol) 服务器配置项目，包含了多个实用的MCP服务器配置。

## 📋 已安装的MCP服务器

### 🎭 Playwright MCP
- **功能**: 网页自动化和测试
- **用途**: 网页截图、表单填写、自动化测试

### 🧠 Sequential-thinking MCP  
- **功能**: 顺序思维分析
- **用途**: 复杂问题分解、逻辑推理

### 🌐 Browser-tools MCP
- **功能**: 浏览器调试和审计
- **用途**: 性能分析、SEO检查、可访问性测试

### 📚 Context7 MCP
- **功能**: 文档检索和库管理
- **用途**: 编程库文档查询、API参考

### 🔄 Interactive-feedback MCP
- **功能**: 交互式反馈系统
- **用途**: 用户反馈收集和处理

### 🐙 GitHub MCP
- **功能**: GitHub API完整访问
- **用途**: 仓库管理、问题跟踪、PR管理、代码搜索

## 🚀 使用方法

1. 确保已安装Node.js和npm
2. 克隆此仓库
3. 配置相应的环境变量（如GitHub令牌）
4. 在支持MCP的AI助手中加载配置

## 📁 文件说明

- `mcp.json` - MCP服务器配置文件
- `package.json` - Node.js依赖配置
- `package-lock.json` - 依赖锁定文件

## 🔧 配置要求

- Node.js 16+
- GitHub个人访问令牌（用于GitHub MCP）
- Python环境（用于Interactive-feedback MCP）
